### Онлайн-институт - MasterVision

## Стек технологий

* ReactJs + Hook
* react-router-dom
* Redux
* Redux thunk
* node-sass
* axios

## Установка
Клонируйте репозиторий.
```
git clone https://github.com/iomp-tech/mv-frontend.git
```

Переход в рабочий каталог.
```
cd mv-frontend
```

Установка зависимостей, запуск сборки.
```
npm install
npm start
```
